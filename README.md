# Интернет-магазин футбольной атрибутики и аксессуаров

Веб-ориентированная информационная система для продажи футбольной атрибутики
и аксессуаров через интернет.

Проект выполнен в рамках курсовой работы по дисциплине «Web-программирование».

---

## Назначение системы

Система предназначена для автоматизации процессов онлайн-продажи товаров
футбольной тематики, включая:

- просмотр каталога товаров;
- формирование корзины;
- оформление заказа;
- хранение информации о товарах и заказах;
- административное управление ассортиментом.

---

## Основной функционал

- Каталог товаров с категориями.
- Страница отдельного товара.
- Корзина покупателя (хранится в сессии).
- Оформление заказа без онлайн-оплаты.
- Административная панель для управления товарами.
- Хранение данных в базе данных PostgreSQL.
- Контейнеризация с использованием Docker.
- Наличие модульных тестов.
- Настроенный CI-процесс.

---

## Используемые технологии

- Python
- Flask
- Flask-SQLAlchemy
- PostgreSQL
- Nginx
- Docker, Docker Compose
- GitHub Actions
- Pytest
- Bootstrap (frontend)

---

## Запуск проекта (локально)

1. Создать виртуальное окружение Python.
2. Установить зависимости из файла `requirements.txt`.
3. Создать файл `.env` на основе `.env.example`.
4. Запустить приложение через `wsgi.py`.

---

## Запуск проекта (Docker)

Для запуска полного стека приложения используется `docker-compose`.
